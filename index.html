Introduction

This is an app so you can stay updated with the school with notifications on dress down days and on school events. Never miss an event or important information.

Features

  • Get notifications about school events.
  • See how many house points the school has.
  • View the school calendar.
  • See our schools tweets and various information.
  • School Bus Cancellations.
  • School map built in.
  • Notepad.

Devices supported

The app is not developed yet but it should be supporting these opperating systems.

  • Any Android device running Ice Cream Sandwich (API 15+).
  • iOS (iPhone)
  • [Later Releases] Windows Phone
